`y = np.array([5, 7, 9, 11, 13])`<br>
`# –æ—Ü–µ–Ω–∫–∞ –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤ –ª–∏–Ω–µ–π–Ω–æ–π —Ä–µ–≥—Ä–µ—Å—Å–∏–∏`<br>
`slope, intercept = np.polyfit(x, y, 1)`<br>

#### –°—Ç–∞—Ç–∏—Å—Ç–∏—á–µ—Å–∫–∏–µ —Ç–µ—Å—Ç—ã (–∫—Ä–∏—Ç–µ—Ä–∏–∏)

###### –î–≤—É—Ö–≤—ã–±–æ—Ä–æ—á–Ω—ã–π t-–∫—Ä–∏—Ç–µ—Ä–∏–π –°—Ç—å—é–¥–µ–Ω—Ç–∞
